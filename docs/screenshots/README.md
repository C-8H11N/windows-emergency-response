# 界面截图

欢迎贡献界面截图！请遵循以下规范：

## 📸 截图规范

| 文件名 | 内容 | 建议尺寸 |
|--------|------|----------|
| `dashboard.png` | 仪表盘主界面 | 1280×800 |
| `scan.png` | 扫描中界面 + 结果 | 1280×800 |
| `findings.png` | 发现项详情展开 | 1280×800 |
| `report.png` | HTML 报告预览 | 1280×800 |
| `themes.png` | 4 种主题对比展示 | 1920×1080 |

## 🎯 截图方式

1. 启动 `win_er_tool.exe`
2. 浏览器访问 `http://127.0.0.1:8000`
3. 按 `F11` 全屏
4. 使用系统截图工具（Win+Shift+S）
5. 保存为 PNG 格式，质量适中
